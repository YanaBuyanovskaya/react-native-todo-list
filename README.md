# Todo List React Native

Домашнє завдання з React Native.

## Функціонал

- Додавання задачі через TextInput
- Відображення всіх задач у списку
- Видалення задачі
- Перевірка на пустий текст
- Alert при пустому полі
- Toast після успішного додавання задачі

## Запуск проєкту

```bash
npm install
npx expo start
